# Car Rental System

## Технологический стек

### Frontend
- **React** + **Vite**  
- **SCSS** 
- **Axios**

### Backend  
- **Node.js** + **Express**  
- **PostgreSQL** 

### Admin

## Установка и запуск

### Запуск фронтенда
```bash
cd client
npm install
npm run dev
```

### Запуск бекенда
```bash
cd server
npm install
npm run server
```

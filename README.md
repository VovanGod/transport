# Car Rental System

## Технологический стек

### Frontend
- **React** + **Vite**  
- **SCSS** для стилей  
- **Axios** для HTTP-запросов  

### Backend  
- **Node.js** + **Express**  
- **PostgreSQL** 
- **CORS** 

## Установка и запуск

### Запуск фронтенда
```bash
cd client
npm install
npm run dev
```

### Запуск бекенда
```bash
cd server
npm install
npm run server
```
